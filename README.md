# GitGrade AI

GitGrade AI is a beginner-friendly project that analyzes a GitHub repository
and converts it into a meaningful Score, Summary, and Personalized Roadmap.

## Problem Statement
Students often do not know how their GitHub projects look to recruiters.
This system evaluates a repository and gives honest feedback.

## Features
- Accepts GitHub repository URL
- Fetches repository data using GitHub API
- Evaluates basic quality indicators
- Generates score, summary, and roadmap

## Tech Stack
- Python
- GitHub REST API

## How to Run
1. Install Python
2. Run the file:
3. Paste a public GitHub repository URL
